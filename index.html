How to connect (manual signaling)

  1. Both peers open this page. Peer A clicks Create Offer.
  2. Peer A copies the Offer SDP and sends it to Peer B (chat, email, etc.).
  3. Peer B pastes the Offer SDP and clicks Create Answer.
  4. Peer B copies the Answer SDP and sends it back to Peer A.
  5. Peer A pastes the Answer SDP and clicks Set Answer.
  6. 🎉 Connection established — video streams appear!
